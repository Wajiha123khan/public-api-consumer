# Public Api Consumer 
A simple python CLI Application  that fecthes country information/data using REST Countries API

## Features
- Search Countries by name
- Displays: 
    - Capitals
    - Population
    - Region 
    - Currency 
    - Language
    - Flag URL

## Installation 
Clone the repository:

```bash
git clone <your-repo-link>
```

Move into project folder:

```bash
cd public-api-consumer
```

Create virtual environment:

```bash
python -m venv venv
```

Activate virtual environment:

Windows:

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Run Project

```bash
python main.py
```
